# 基本高精度加法
这个cpp程序提供了一个基本的高精度加法实现。

## 高精度是什么？
比如说当两个爆了int64_t的数相加的时候，一般数据都会溢出而使计算结果错误；但我们人还是能手算的~~不过能累死人~~。为此我们让计算机模拟人手算加法进位的过程。  
~~我的计算机加法不可能这么拟人~~

## 怎么实现的？
两个数组相加，再处理进位。
